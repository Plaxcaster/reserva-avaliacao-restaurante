# reserva-avaliacao-restaurante
Repositório para o TC3


### Para rodar o projeto:

Versão desenv, utilizando banco de dados h2:

```

./mvnw spring-boot:run

```

### Tecnologias utilizadas

- Spring-boot

- Swagger

- Banco de dados h2

- Dependência de testes Jacoco

- Docker

- Deploy via Render
